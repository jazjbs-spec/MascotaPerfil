* 🐶 TARJETA DE PERFIL DE MASCOTA

Aplicación Android desarrollada con **Jetpack Compose** enfocada en la construcción de interfaces visuales modernas mediante el uso de composables, layouts y componentes estilizados.

---

* ✨ DESCRIPCIÓN

La aplicación muestra una tarjeta de presentación visual de una mascota utilizando una imagen de fondo, textos descriptivos y una estructura organizada con layouts de Compose.

El objetivo principal es practicar la creación de interfaces dinámicas y visualmente atractivas en Android Studio.

---

* 🎯 OBJETIVOS IMPLEMENTADOS

- Construcción de UI con **Jetpack Compose**
- Uso de estructuras visuales (`Box` y `Column`)
- Manejo de imágenes y superposición de elementos
- Aplicación de alineación y espaciado
- Personalización de texto y estilos visuales
